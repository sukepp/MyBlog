The fast sort

1. Definition:

Quick sorting, which is also the most commonly used sorting algorithm in practice, is fast and efficient. Just like the name, quick sorting is one of the best sorting algorithms.

2. Algorithm description:

The idea of quick sorting is the idea of divide and conquer. ...

Quick sorting is to find an element (in theory, you can find one) as a reference, and then partition the array so that the value of the element on the left side of the reference is not greater than the reference value, and the value of the element on the right side of the reference is not less than the reference value. Thus, the element as a reference is adjusted to the correct position after sorting. Recursive quick sorting, adjusting other n-1 elements to the correct position after sorting. Finally, each element is in the correct position after sorting, and the sorting is completed.

Therefore, the core algorithm of the quick sort algorithm is the partition operation, that is, how to adjust the position of the reference and adjust the final position of the return reference to divide and recur.

3. Code implementation:

    #include <stdio.h>
    #include <stdlib.h>
    #include <iostream>

    using namespace std;

    void quick_sort(int a[], int left, int right)
    {
        int i, j, pivot, temp;

        if(left >= right)
            return;

        i = left;
        j = right;
        pivot = a[left];

        while(i < j) {
            while(a[j] >= pivot && i < j) { 
                j--;
            }
            while(a[i] <= pivot && i < j) {
                i++;
            }
            temp = a[i];
            a[i] = a[j];
            a[j] = temp;
        }

        a[left] = a[i];
        a[i] = pivot;
        quick_sort(a, left, i - 1);
        quick_sort(a, i + 1, right);
    }

    void Quick_Sort(int a[], int n)
    {
        quick_sort(a, 0, n - 1);
    }

    int main()
    {
        int i;
        int a[] = {5, 4, 3, 9, 8, 7, 6, 2, 1, 0};

        for(i = 0; i < 10; i++)
            printf("%d ", a[i]);
        printf("\n");

        Quick_Sort(a, 10);

        for(i = 0; i < 10; i++)
            printf("%d ", a[i]);
        printf("\n");

        system("pause");
        return 0;
    }
